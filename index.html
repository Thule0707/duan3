Trường hợp 1:

git init: khởi tạo

git add .: add toàn bộ dự án vào trong staging area (vùng chờ commit xác nhận)

git add ten_thu_muc: add lẻ thư mục, tệp

git commit -m "first commit": sau khi commit, code đẩy vào kho lưu trữ và đẩy lên github

git branch -M main: chuyển từ master(máy chủ git) sang nhánh main

git remote add origin https//github.com/.../...:remote đến máy chủ git

git push -u origin main: đẩy code lên máy chủ github
